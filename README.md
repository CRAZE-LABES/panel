## 🛠️ Installation Guide

### ✅ System Preparation

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install -y curl software-properties-common ca-certificates
```
⚙️ Install Node.js 20
```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# Check version
node -v
npm -v
```

---

🔧 CrazePanel Installation

git clone https://github.com/craze-labes/panel
cd panel
npm install
npm run seed
npm run createUser
node .


---

🛠 CrazeDaemon Installation

git clone https://github.com/craze-labes/daemon
cd daemon
npm install
